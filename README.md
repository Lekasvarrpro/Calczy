# Calczy – Interactive Web Calculator

## 1. Overview

Calczy is a modern, interactive web calculator built with HTML, CSS, and JavaScript. It features a neon-inspired UI, responsive design, smooth micro-interactions, and a live calculation history, making it ideal for showcasing front-end skills and polished UI design.

## 2. Tools Used

HTML5 – structure and semantic markup

CSS3 – modern styling, animations, and responsive layout

JavaScript (ES6) – dynamic calculations, live preview, and history logging

Git & GitHub – version control and deployment

## 3. Working

Enter numbers and operators using keyboard or buttons.

Results are displayed live as you type.

Press = or Enter to calculate the result.

Use C to clear, Del/Backspace to remove digits, and long-press support is included.

Previous calculations are automatically saved in the history log.

Results can be copied to clipboard with a single click.

## 4. Features

✅ Live Expression Preview — see results as you type

✅ Keyboard Support — type 0-9, operators, Enter, Backspace, C

✅ Smart Multiplication — handles 2(3) or (4)(5) as valid expressions

✅ Long Press Support — hold down buttons like Del or numbers

✅ History Log — automatically logs previous calculations

✅ Copy Result to Clipboard

✅ Responsive Design — works great on mobile and desktop

✅ Styled Scrollbars — clean and modern look

## 5. Conclusion

Calczy combines clean design, interactivity, and functionality, making it a portfolio-ready project that demonstrates front-end development skills and UI/UX sensibilities.
